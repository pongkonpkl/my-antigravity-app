# 🌍 วิธีการทำให้เว็บไซต์ออนไลน์ (Deploy to Cloud)

เมื่อคุณอัปโหลดไฟล์ขึ้น GitHub เสร็จแล้ว **นั่นเป็นเพียงการเก็บโค้ดไว้ในตู้เซฟครับ** ตัวเว็บไซต์ยังไม่ได้เริ่มทำงานจริงๆ (เหมือนเราซื้อเครื่องเล่นเกมมาแต่ยังไม่ได้เสียบปลั๊ก)

เพื่อให้ทุกคนเข้าชมเว็บไซต์ได้จากทั่วโลก คุณต้องทำขั้นตอนสุดท้ายคือ **"การเสียบปลั๊ก" (Deploy)** ครับ:

### ขั้นตอนที่ 1: เชื่อมต่อกับ Streamlit Cloud (ฟรี)
1.  ไปที่เว็บไซต์ **[share.streamlit.io](https://share.streamlit.io)**
2.  กดปุ่ม **"Continue with GitHub"** เพื่อล็อกอินด้วยบัญชีที่คุณเพิ่งอัปโหลดโค้ดไป
3.  เมื่อเข้าไปแล้ว ให้กดปุ่ม **"Create app"** (หรือ "New app") สีฟ้าขวามือ

### ขั้นตอนที่ 2: เลือก Repository ของคุณ
1.  ในช่อง **"Repository"**: ให้เลือกชื่อโปรเจกต์ที่คุณเพิ่งสร้างใน GitHub
2.  ในช่อง **"Main file path"**: พิมพ์คำว่า `app.py`
3.  กดปุ่ม **"Deploy!"**

### ขั้นตอนที่ 3: รอให้ระบบติดตั้ง
1.  หน้าจอจะเปลี่ยนเป็นรูปการ์ตูนกำลังทำอาหาร (Cooking) หมายความว่าระบบกำลังติดตั้ง Library ที่ผมเขียนไว้ใน `requirements.txt` ให้คุณครับ
2.  รอประมาณ 2-3 นาที คุณจะได้ URL เว็บไซต์ที่เป็นชื่อโปรเจกต์ของคุณ ซึ่ง **สามารถแชร์ให้เพื่อนๆ ทั่วโลกเปิดดูได้ทันทีครับ!**

---

### 💡 ข้อควรทราบ (ความลับของระบบ)
- **Backend (API)**: โปรเจกต์นี้มีตัว Backend (`main.py`) ด้วย ซึ่งหากคุณ Deploy แค่ `app.py` ระบบ Social Feed อาจจะยังใช้งานไม่ได้ 100% จนกว่าจะนำ `main.py` ไปรันบน Cloud ตัวอื่น (เช่น Render หรือ Railway)
- หากคุณต้องการให้ทุกอย่างทำงานสมบูรณ์แบบที่สุดบน Cloud ตัวเดียว แนะนำให้ในอนาคตเรายุบรวมทั้ง 2 ตัวเข้าด้วยกันครับ

### ⚠️ สำคัญมาก: ความแตกต่างของ URL
- **GitHub Pages** (`github.io`): สำหรับเว็บไซต์ HTML ธรรมดา (รัน Python ไม่ได้ จึงเห็นเป็นหน้าจอขาว)
- **Streamlit Cloud** (`streamlit.app`): สำหรับแอป AI และ Dashboard แบบโปรเจกต์นี้ (รัน Python ได้)

ตอนนี้ลองทำตาม **ขั้นตอนที่ 1** ดูก่อนนะครับ เว็บไซต์สวยๆ ของคุณกำลังจะออนไลน์แล้ว! 🚀
